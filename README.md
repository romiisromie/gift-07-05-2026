# 🌸 Our Story Timeline

A special digital gift created for May 7th. This project features an interactive Japanese Sakura Bonsai tree with falling heart-sparks and a live counter of our shared history.

## ✨ Features
- **Interactive Sakura:** Animated bonsai tree that blooms on click.
- **Sparkle Animation:** Floating heart-shaped petals.
- **Live Timer:** Real-time counter showing days, hours, and seconds.
- **Timeline:** Key milestones of our journey.

## 🛠️ Built With
- HTML5
- CSS3 (Animations & SVG)
- JavaScript
